# Note
將執行清單化為不同層次的項目:
- 5樓 : 目的及原則
- 4樓 : 願景
- 3樓 : 目標
- 2樓 : 關注點及責任範圍
- 1樓 : 項目
- 地面 : 行動

## 地面 : 行動
目前要完成的行動，也就是清單中的行動

## 1樓 : 項目
指項目清單中的項目，這些項目底下包含了許多的小代辦清單

## 2樓 : 關注點及責任範圍
可以再將這些項目進一步分類為不同的領域，如:
- 工作
- 家庭
- 健康
- 財務
- 長期規劃
- 配偶
- 自我提升

## 3樓 : 目標
未來1~2內的目標

## 4樓 : 願景
3~5年內環境、家庭、事業的發展

## 5樓 : 目的及原則
通常為較為抽象的思考層次
- 自己的人生目標到底是甚麼?
- 生而為人的願景是甚麼?

## 注意事項
- 作者建議在執行清單時應該要由下而上，先從最底層的類別開始，原因是因為只有當底層的事件被妥善處理時才會有心力去處理更高層的思考
