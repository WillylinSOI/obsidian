# Note
將清單編列好後下一步就是執行清單，清單可以用以下幾種方法執行:
- 四標準法 : 根據情境、有多少時間、精力、重要性分類
- 三分類法 : 根據已安排工作、突發工作、自己的工作分類
- 六層次法 : 根據原則、願景、目標、責任、項目、行動分類

## 注意事項
- 一次只注意處裡一件事情，多工只會分心
- 不要依照重要性去處理事件，採用先進先出或是後進先出原則
- 決定要處裡的事項就處理到好，不要處理到一半後又放回清單中
