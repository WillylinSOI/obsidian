# Note
![](Pasted%20image%2020230128214618.png)
當出現一件未完成的代辦事項時，腦中會不斷的提醒自己導致自身的專注力下降，而GTD方法最主要的目標在於清空大腦中的代辦事項，將每一件事情來源都化為實際得行動:
- 蒐集 : 蒐集生活中的事件或是行動
- 清單 : 針對不同的情況建立不同的解決清單
- 執行 : 根據適當的時機地點執行清單中的事項

## 注意事項
- 建構GTD系統時要讓每個步驟執行起來都盡可能的簡單，這樣才不會對GTD系統感到厭煩
