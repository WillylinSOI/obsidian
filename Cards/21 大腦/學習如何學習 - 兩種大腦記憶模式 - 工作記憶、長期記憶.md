# Note
大腦同時存在兩種不同的記憶系統，長期記憶和工作記憶
- [[學習如何學習 - 長期記憶]] : 遠期記憶，只記住真正重要的事情，睡眠時會鞏固這區域的記憶內容
- [學習如何學習 - 工作記憶](學習如何學習%20-%20工作記憶.md) : 短期記憶，短期需要但並不需要保留至長期記憶中
![](Extras/Media/image/S__9412624.jpg)
當大腦在運作時就會將所需資料從長期記憶中抓取到工作記憶中
