# Alternate Checkboxes

> Different checkbox types

##

Checkbox Types

```markdown
- [x] Unchecked
- [x] Regular
- [X] Checked
- [-] Dropped
- [>] Forward
- [D] Defer
- [?] Question
- [/] Half Done
- [+] Add
- [R] Research
- [!] Important
- [i] Idea
- [B] Brainstorm
- [P] Pro
- [C] Con
- [Q] Quote
- [N] Note
- [b] Bookmark
- [I] Information
- [p] Paraphrase
- [L] Location
- [E] Example
- [A] Answer
- [r] Reward
- [c] Choice
```
